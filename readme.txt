Miembros del equipo: Fidel Lascano.
Descripción del proyecto: El proyecto consiste en una aplicación web, que permite al usuario manejar sus finazas personales.
El usuario puede crear una cuenta, iniciar sesión, agregar sus ingresos y gastos, y ver un resumen de su situación financiera.
La aplicación también ofrece gráficos y estadísticas para ayudar al usuario a entender mejor sus finanzas.
Y para la version de pago la aplicacion ofrece la funcion de compartir gastos y ver una vista combinada de las finanzas de dos o mas usuarios.
Tecnologías utilizadas: HTML, CSS, JavaScript.
